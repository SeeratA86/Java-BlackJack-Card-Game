# JavaFX-BlackJack-Card-Game
## Overview:
JavaFX card game application demonstrating object-oriented programming, MVC structure, FXML views, controlls, and game state management.

## Features
- JavaFX graphical user interface
- Login/player setup screen
- Dealer and player game views
- Card and deck management
- Blackjack game logic
- MVC-based project structure
- FXML views and controller classes

## Technologies Used
- Java
- JavaFX
- FXML
- CSS
- Object-Oriented Programming
- MVC Architecture

## Project Structure
- `src/model` contains the game logic classes
- `src/controller` contains JavaFX controller classes
- `src/view` contains the FXML and CSS files
- `src/au/edu/uts/ap/javafx` contains helper JavaFX classes used to load views
